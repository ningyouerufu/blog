利用git将项目部署到github的方法

1.从github上找到自己的项目地址

2.在本地创建项目文件夹并用git克隆项目到项目文件夹

创建文件夹后鼠标右键点击git bash here.

输入 

$git clone https://github.com/XXX/XXX.github.io.git

(网页链接为自己项目地址)

3.将克隆文件夹内除.git文件夹外的文件替换为所需文件

4.程序内执行代码检查文件增减

替换文件以后用程序cd进这个文件夹并输入$git add .

(注意add后面有个点)

然后输入$ git commit -m "test"

(test可随意输入)

5.部署项目

输入$ git push -u origin master

即可部署成功
